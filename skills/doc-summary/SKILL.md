## 職責
- 根據 doc/ 裡面的資料，summary近期的改動，把資料更新到 doc/summary
- 參考 git 把資料更新到 doc/summary
- 維護 Claude.md 或 AGENTS.md 讓他保持索引的角色輕量化，需要細節請讓他到 doc/ 查看

## doc/ 資料存放結構
檔案結構可以自己調整，單一檔案不要太重為主
我要在 github 上可以閱讀，可以有一些超連結輔助
- architectures: 程式架構
- features: 各功能的相關說明
- utils: 小工具的存放
- 其他...需要的分類
